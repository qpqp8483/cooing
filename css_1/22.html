IS(Image sprite)기법

  • 과거에는 이미지용량이 우선시되서 무조건 작은 파일로 각각 저장을 사용
  • 현재는 이미지용량(다운로드 속도)보다 브라우저 렌더링 속도를 우선시 해서 이미지를 한개로 사용
  • 이미지 원본
  • IS기법 적용

IR(Image Replacement, 이미지 대처기법)

  • 주로 웹표준 코딩에서는 구조와 디자인 서로 다른경우에 사용
  • 마크업에서의 텍스트를 CSS에서 이미지로 바꿔 사용하는 기법
  • 예)네이버에서 h1에 '네이버' 라고 되어 있지만 실제론 naver로고 이미지가 적용
