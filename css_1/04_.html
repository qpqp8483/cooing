블록요소 vs 인라인요소

블록요소

p, div, ol, dl, h1~h6 처럼 박스형태로 화면폭을 가지는 공간지정 태그
- 기본 폭값이 100%
- margin값을 가질 수 있음
- z-index(레이어) 값을 가질 수 있음
- position 값 지정이 가능

인라인요소

글자, span, strong, time, date... 와 같이 글자처럼 정렬되는 공간지정 태그
- 기본 정렬은 text-align을 이용
- padding, margin등 여백관련 속성이 잘 적용되지 않음
- 크기 지정이 되지 않는다
- 인라인요소를 블록요소처럼 정렬하기 위해서는 display속성으로 블록으로 변경하여 사용

인라인블록요소

인라인요소처럼 정렬하면서 블록의 속성을 사용할 수 있는 요소
- 블럭요소 처럼 크기를 갖고 위치를 가짐
- 인라인요소 처럼 글자정렬이 가능
- table, img...
