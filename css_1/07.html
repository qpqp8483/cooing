input의 종류

text : 글자 입력폼

hidden : 정보값을 숨겨 고정

maxlength : 글자수를 제한시키는것

disabled : 글자 입력(수정)을 제한시키는것

readonly : 글자 입력(수정)을 제한시키는것

password: 비밀번호 입력폼

checkbox: 다중 선택을 할때 쓰는 폼

radio: 한 개만 선택을 할때 쓰는 폼



file : 파일첨부를 할 때

submit : 전송/제출/완료/ok 버튼으로 form문에 action이 실행

reset : 입력한 정보를 초기화, 다시쓰기 기능

image + src : 버튼을 이미지로 대체

butten : 버튼

butten tag : 버튼
