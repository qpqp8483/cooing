기타 CSS선택자

  • 전체선택자 : *
  • 자손선택자 : #wrap ul
  • 자식선택자 : ul>li
  • 형제선택자 : ul+a(ul을 바로 앞에둔 a태그만 선택)
  • 형제선택자 : ul~a (ul이후에 모든 a태그 형제만 선택)
  • 그룹선택자 : ul, a
  • 가상클래스 : link, visited, hover, active
  • 가상클래스 : nth-child(), first-child, last-child, first-letter, first-line nth-child(2n), nth-child(odd), nth-child(even), focus, checked, [href=http://naver.com] not(#wrap),
